# anavanzin.com

Site estático (duas páginas) para o GitHub Pages.

- index.html — a mesa (desktop) · página inicial
- conceitos.html — página de conceitos da tese
- ícone "citações" abre /quotes/ (pasta que já existe no seu repo)

PT/EN (botão no topo direito) e layout mobile próprio.

## Publicar (no repo que já serve o domínio: anavvanzin.github.io)
1. github.com/anavvanzin/anavvanzin.github.io → Add file → Upload files.
2. Arraste TODOS os arquivos desta pasta (incl. assets/, .nojekyll, CNAME).
   Substitui o index.html placeholder pela mesa. NÃO apague a pasta quotes/.
3. Commit changes. Em ~1 min, anavanzin.com mostra o novo site.

.nojekyll é essencial (sem ele o GitHub ignora _ds_bundle.js).
Fontes vêm do Google Fonts (precisa de internet).
Capítulos por inferência: iconologia/iconometria=Cap.3, atlas=Cap.4, economia=Cap.2.
